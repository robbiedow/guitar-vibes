# ToneBoard

Personal guitar amp preset builder for the Fender Mustang Micro Plus. PWA — add to your home screen for a native app experience.

## Stack

- Next.js 15 + React 19
- Tailwind CSS 4
- localStorage for persistence
- PWA with service worker

## Setup

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub, connect repo to [Vercel](https://vercel.com), deploy. Zero config needed.
